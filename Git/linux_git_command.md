# linux 명령어 모음

1. **cd**  -  "change directory" 

* 현재 디렉토리 변경

```bash
$cd directory_name
```

2. **pwd**  -  "print working directory"

* 현재 작업중인 디렉토리 정보 출력

```bash
$pwd
/home/test
```

3. **ls**  -  "list"

* 디렉토리 목록 확인
```bash
$ls
testfile1 testfile2 testfile3

$ls -a
./ ../ testfile1 testfile2 testfile3

$ls -al

```

* ls -a 옵션 (all)  -  **숨겨진** 파일이나 디렉토리도 보여줌
